## Projetos em Destaque

### 1. TPG Insight AI (MVP de Modernização e Governança) - Ford Motor Company
* **O Problema:** Descentralização de laudos técnicos e validação veicular. O uso de canais não oficiais (WhatsApp) gerava riscos graves de vazamento de dados sensíveis.
* **A Solução:** Idealizei e desenvolvi do zero, em 14 dias, um MVP (SPA/PWA) integrado ao Firebase. Criei o "Smart Laudo", integrando LLMs via API com Engenharia de Prompt e higienização de inputs (*Security by Design*).
* **O Resultado:** Eliminação do trânsito de dados em canais inseguros e automação da digitação, provando o valor da IA para a liderança técnica (PIM).
* **Stack:** `JavaScript (ES6+)` `Firebase` `Integração de APIs (LLM)` `HTML/CSS`
